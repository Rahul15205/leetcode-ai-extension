# Privacy Policy for LeetCode AI Assistant

Last updated: January 11, 2025

## Overview

LeetCode AI Assistant is a Chrome extension that provides AI-powered coding assistance for LeetCode problems. This privacy policy explains how we collect, use, and protect your information.

## Information We Collect

### Personal Information
- **API Key**: You provide a Groq API key which is stored locally in your browser's secure storage
- **No Personal Data**: We do not collect names, email addresses, or other personal information

### Usage Data
- **Problem Content**: When you use the extension, the current LeetCode problem and your code (if submitted for review) are sent to Groq's API for analysis
- **No Analytics**: We do not track usage patterns, collect analytics, or store interaction data

## How We Use Information

### API Key Storage
- Your Groq API key is stored locally in Chrome's secure storage
- The API key never leaves your browser except for authenticated requests to Groq
- We do not have access to your API key

### AI Processing
- Problem descriptions and code are sent to Groq's AI service for analysis
- This data is used solely to provide AI-powered assistance
- We do not store or retain any problem content or code

## Data Sharing

### Third-Party Services
- **Groq**: Problem content and code are sent to Groq's API for AI processing
- **No Other Sharing**: We do not share data with any other third parties

### Data Storage
- All data is processed in real-time and not stored by our extension
- Your API key is stored only in your local browser storage
- No data is transmitted to our servers (we don't have any)

## Data Security

### Local Storage
- API keys are stored using Chrome's secure storage API
- Data is encrypted and isolated to your browser profile

### Transmission Security
- All communication with Groq uses HTTPS encryption
- No data is transmitted over unencrypted connections

## Your Rights

### Data Control
- You can delete your API key at any time through the extension settings
- You can uninstall the extension to remove all locally stored data
- You control what code and problems are sent for analysis

### Groq Data Policy
- Data sent to Groq is subject to [Groq's Privacy Policy](https://groq.com/privacy-policy/)
- You can review Groq's data handling practices on their website

## Children's Privacy

This extension is not intended for use by children under 13. We do not knowingly collect personal information from children under 13.

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted in this document with an updated revision date.

## Contact Information

For questions about this privacy policy, please:
- Open an issue on our GitHub repository
- Contact us through the Chrome Web Store developer contact

## Compliance

This extension complies with:
- Chrome Web Store Developer Program Policies
- General Data Protection Regulation (GDPR)
- California Consumer Privacy Act (CCPA)

## Data Retention

- We do not retain any user data
- All processing is done in real-time
- Local storage (API key) is retained until you delete it or uninstall the extension
